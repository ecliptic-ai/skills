# skills

A Claude Code plugin marketplace from [Ecliptic-AI](https://github.com/ecliptic-ai).

## Add the marketplace

```sh
/plugin marketplace add ecliptic-ai/skills
```

## Plugins

| Plugin | Description |
| :----- | :---------- |
| [`cadence`](./plugins/cadence) | AI-augmented video splicing framework — sensors, shared timeline, renderer. TypeScript MCP server with a Python librosa sidecar for beat detection. |
| [`ecliptic`](./plugins/ecliptic) | AI-powered beat-synced video editing using Gemini and FFmpeg. |

## Install a plugin

```sh
/plugin install cadence@skills
/plugin install ecliptic@skills
```

See [Discover and install plugins](https://docs.claude.com/en/discover-plugins) for scopes and management commands.
